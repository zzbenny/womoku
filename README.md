# 五子棋游戏

这是一个使用 Python 和 Pygame 开发的五子棋游戏。

## 安装依赖

在运行游戏之前，请先安装必要的依赖：

```bash
pip install -r requirements.txt
```

## 运行游戏

安装完依赖后，直接运行 main.py 文件即可开始游戏：

```bash
python main.py
```

## 游戏规则

1. 黑子先手，双方轮流下棋
2. 在空位置点击鼠标左键放置棋子
3. 任意方向（横、竖、斜）连成5个或以上棋子即获胜
4. 游戏结束后会显示获胜方

## 操作说明

- 使用鼠标左键点击棋盘交叉点放置棋子
- 关闭窗口退出游戏
